## Installation

You can install this plugin using your favorite package manager.  
For example, if you use dein, execute

```vim
call dein#add('keiyakeiya/PapilioDehaanii.vim')
```

## Usage

```vim
colorscheme papilio_dehaanii
```

```vim
AirlineTheme papilio_dehaanii
```
